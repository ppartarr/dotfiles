## Install
```bash
git clone https://github.com/ppartarr/dotfiles ~/.dotfiles
cd ~/.dotfiles
```

### Usage
```bash
# create symlinks
$ chmod +x symlink && ./symlink

# remove them
$ chmod +x rm-symlink && ./rm-symlink
```
