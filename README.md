## Install
```bash
git clone https://github.com/ppartarr/dotfiles ~/.dotfiles
cd ~/.dotfiles
sh install.sh
```

## Uninstall
```bash
cd ~/.dotfiles && sh uninstall.sh
cd && rm -rf ~/.dotfiles
```
